# 📄 Resume Analyzer & Job Matcher

An AI-based Resume Analyzer that extracts skills from PDF resumes and recommends suitable job roles using Machine Learning and NLP techniques.

---

## 🚀 Features
- Upload single or multiple resume PDFs
- Extracts skills from resumes
- Resume score out of 100
- Job role matching using TF-IDF & cosine similarity
- Skill gap analysis
- Visual job match comparison
- Interactive Streamlit web application

---

## 🛠 Tech Stack
- Python 3
- Streamlit
- Pandas
- Scikit-learn
- PyPDF2
